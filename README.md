### 01 - Sequential Structure

Basic programs executed step by step, without conditions or loops.
Exercises

1. arithmetic_mean.c → Calculates the average of two numbers
2. fahrenheit_converter.c → Converts Celsius to Fahrenheit
3. basic_math_operations.c → Performs basic arithmetic operations
4. heron_triangle_area.c → Calculates triangle area using Heron’s formula
5. car_efficiency_calculator.c → Calculates fuel efficiency
6. best_value_calculator.c → Compares product values
7. fruit_collection_time.c → Calculates time based on conditions
8. seconds_to_hms.c → Converts seconds to hours, minutes, and seconds




### 02 - Decision Structures

Programs that use conditional logic (if, else, else if) to make decisions.
Exercises

1. grade_average_status.c → Calculates the student’s average and determines academic status
2. divisibility_check.c → Checks if a number is divisible by another
3. common_divisibility.c → Checks if two numbers share divisibility conditions
4. number_sign.c → Determines if a number is positive, negative, or zero
5. basic_calculator.c → Performs basic arithmetic operations
6. allowance_comparison.c → Compares two values to check if they are equal
7. equal_values_count.c → Counts how many values are equal among three inputs
8. maximum_value.c → Finds the largest value among given numbers
9. harry_potter_quiz.c → A multiple-choice quiz about Harry Potter
10. salary_increase.c → Calculates a salary increase based on conditions
11. zero_or_one_game.c → Determines the winner of the “zero or one” game
12. price_guessing_game.c → Simulates a price guessing competition
13. price_guessing_game_v2.c → Improved version of the price guessing game
14. student_grade_evaluation.c → Evaluates student performance based on grades
15. work_schedule_check.c → Checks if someone should be working at a given time
16. rock_paper_scissors.c → Simulates the rock-paper-scissors game
17. even_odd_game.c → Determines the winner based on even or odd results
18. triangle_classification.c → Classifies a triangle by its sides
19. value_ordering.c → Orders values based on their magnitude
20. quadratic_equation_solver.c → Solves quadratic equations using Bhaskara formula
21. fruit_collection_time.c → Calculates time required to collect fruits
22. zero_or_one_american.c → A multi-player variation of the zero or one game
