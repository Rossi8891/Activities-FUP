# FUP C EXERCISES

This repository contains my solutions to programming exercises developed during the Fundamentals of Programming (FUP) course

---

## About

The goal of this repository is to build a strong foundation in programming logic using the C language, progressing from basic concepts to more advanced problem-solving techniques

### 01 - Sequential Structure

Basic programs executed step by step, without conditions or loops.
Exercises:

1. arithmetic_mean.c → Calculates the average of two numbers
2. fahrenheit_converter.c → Converts Celsius to Fahrenheit
3. basic_math_operations.c → Performs basic arithmetic operations
4. heron_triangle_area.c → Calculates triangle area using Heron’s formula
5. car_efficiency_calculator.c → Calculates fuel efficiency
6. best_value_calculator.c → Compares product values
7. fruit_collection_time.c → Calculates time based on conditions
8. seconds_to_hms.c → Converts seconds to hours, minutes, and seconds




### 02 - Decision Structures

Programs that use conditional logic (if, else, else if) to make decisions.
Exercises:

1. grade_average_status.c → Calculates the student’s average and determines academic status
2. divisibility_check.c → Checks if a number is divisible by another
3. common_divisibility.c → Checks if two numbers share divisibility conditions
4. number_sign.c → Determines if a number is positive, negative, or zero
5. basic_calculator.c → Performs basic arithmetic operations
6. allowance_comparison.c → Compares two values to check if they are equal
7. equal_values_count.c → Counts how many values are equal among three inputs
8. maximum_value.c → Finds the largest value among given numbers
9. harry_potter_quiz.c → A multiple-choice quiz about Harry Potter
10. salary_increase.c → Calculates a salary increase based on conditions
11. zero_or_one_game.c → Determines the winner of the “zero or one” game
12. price_guessing_game.c → Simulates a price guessing competition
13. price_guessing_game_v2.c → Improved version of the price guessing game
14. student_grade_evaluation.c → Evaluates student performance based on grades
15. work_schedule_check.c → Checks if someone should be working at a given time
16. rock_paper_scissors.c → Simulates the rock-paper-scissors game
17. even_odd_game.c → Determines the winner based on even or odd results
18. triangle_classification.c → Classifies a triangle by its sides
19. value_ordering.c → Orders values based on their magnitude
20. quadratic_equation_solver.c → Solves quadratic equations using Bhaskara formula
21. fruit_collection_time.c → Calculates time required to collect fruits
22. zero_or_one_american.c → A multi-player variation of the zero or one game



### 03 - Repetition Structures

Exercises involving loops such as for, while, and do while.
Exercises:

1. Repeat_print_phrase.c: Repeats and prints a phrase multiple times based on user input
2. Sum_range_a_to_b.c: Calculates the sum of numbers within a given range
3. Even_sum_range.c: Computes the sum of even numbers in a specified range
4. Even_divisible_by_3_sum.c: Sums even numbers that are divisible by 3
5. Even_divisible_by_3_and_x_sum.c: Sums even numbers divisible by both 3 and another value
6. Average_from_inputs.c: Calculates the average of multiple input values
7. Count_occurrences.c: Counts how many times a specific value appears.
8. Fit_capacity_checker.c: Checks if values fit within a defined capacity
9. Minimum_value_finder.c: Finds the smallest value among inputs
10. MinMax_sum_calculator.c: Calculates the sum between minimum and maximum values
11. Parity_force_analyzer.c: Analyzes numbers based on parity conditions
12. Odd_positive_average_calculator.c: Computes the average of odd positive numbers
13. Digit_counter.c: Counts the number of digits in a number
14. Water_billing_system.c: Simulates a water billing calculation system
15. Frequency_counter.c: Counts frequency of values in a sequence
16. Blackjack_score.c: Calculates Blackjack scores based on input values
17. Palindrome_validator.c: Checks if a number is a palindrome



### 04 - Subroutines

Programs using functions and modular programming concepts.
Exercises

1. Factorial_difference.c: Calculates the difference between the factorials of two numbers using functions
2. Expression_calculator.c: Evaluates mathematical expressions using functions to organize operations
3. Fahrenheit_to_celsius.c: Converts temperature from Fahrenheit to Celsius using a dedicated function
4. Point_distance_calculator.c: Calculates the distance between two points in a 2D plane using a function
5. Euler_series_approximation.c: Approximates Euler’s number (e) using a series expansion with functions
6. Prime_number_checker.c: Checks whether a number is prime using a function
7. Square_root_calculator.c: Calculates the square root of a number using a function-based approach
8. Digit_counter.c: Counts the number of digits in an integer using a function
9. Triangle_generator.c: Generates a numeric or character-based triangle pattern using functions
10. Triangle_generator_and_sum.c: Generates a triangle and calculates the sum of its elements using functions
